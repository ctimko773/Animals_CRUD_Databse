Welcome to the Animals API!

This API allows you to manage a collection of animals. You can create, read, update, and delete animal records.

Endpoints:

  • POST /items: Create a new animal record.
  • GET /items: Retrieve all animal records.
  • GET /items/<id>: Retrieve a specific animal record by ID.
  • PUT /items/<id>: Update a specific animal record by ID.
  • DELETE /items/<id>: Delete a specific animal record by ID.
